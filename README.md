

# Hotel Reservation System

This project is a simple Hotel Reservation System implemented in Java. It allows customers to register, login, book rooms, cancel bookings, update their data, and delete their accounts. It also provides functionalities for administrators to manage hotels, rooms, customers, and admins.

## Features

- Customer Registration: New customers can register by providing their details such as name, age, phone number, email, address, and password.
- Customer Login: Registered customers can log in using their email and password.
- Room Booking: Customers can browse available rooms and book a room of their choice.
- Cancellation: Customers can cancel their bookings if needed.
- Data Update: Customers can update their personal information like name, age, phone number, and address.
- Account Deletion: Customers can delete their accounts if they wish to.
- Admin Functions: Administrators have functionalities to add hotels, add rooms to hotels, delete rooms from hotels, display all rooms in a hotel, display all customers, and display all admins.
